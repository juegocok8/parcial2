📝 Enunciado: Aplicación para practicar tablas de multiplicar con React

🎯 Requerimientos funcionales

1. Selección de tablas
Los usuarios podrán elegir la tabla o tablas que desean practicar antes de iniciar. Debe incluir imágenes de números con figuras llamativas (números animados).
2. Orden aleatorio sin repetición
El orden de los números del 1 al 10 tiene que ser al azar sin repetirse (Fisher-Yates shuffle).
3. Formato de preguntas
Las preguntas se mostrarán en el formato: 5 x 7 = _____
4. Mensajes generados por IA para respuestas correctas
Los comentarios para respuesta correcta deben ser generados por IA (usando API de OpenAI, Gemini, o simulando con un array de mensajes creativos estilo IA). Ejemplos:
✓ ¡Excelente! Eres un genio de las matemáticas ✓ ¡Perfecto! Sigue así campeón ✓ ¡Muy bien! Tu esfuerzo está dando frutos ✓ ¡Increíble! Lo hiciste mejor que una calculadora
5. Mensajes generados por IA para respuestas incorrectas
Los comentarios para respuesta incorrecta deben ser generados por IA, motivadores y alentadores. Ejemplos:
✗ ¡Casi! Revísalo de nuevo, tú puedes ✗ No te rindas, cada error te acerca al éxito ✗ Intenta de nuevo, la práctica hace al maestro ✗ ¡Sigue intentando! Los grandes matemáticos también se equivocan
6. Generación aleatoria de mensajes IA
Cada vez que el usuario responda, se debe seleccionar aleatoriamente un mensaje del array de mensajes estilo IA (simulando variedad y personalidad).
7. Flujo del ejercicio
El componente React generará cada una de las preguntas de la(s) tabla(s) seleccionada(s) y esperará la respuesta del usuario.
8. Resultados finales
Al finalizar, verificará:
  • ✅ Porcentaje de aciertos
  • ✅ Porcentaje de errores
  • ✅ Número total de respuestas correctas e incorrectas
9. Retroalimentación visual con IA
  • Si el porcentaje es mayor o igual al 75%: mostrar una carita feliz animada y un mensaje de felicitación generado por IA.
  • Si el porcentaje es menor al 75%: mostrar una imagen animada estudiando y un mensaje de ánimo generado por IA para seguir practicando.
