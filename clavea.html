📌 Tecnologías requeridas:

⚛️ React 📦 useState / useContext 🎨 CSS Modules / Tailwind
No se permite el uso de base de datos · Los datos se manejarán con estado local de React (arrays/objetos en el frontend).

📝 Enunciado: Crear un carrito de compras con React

🎯 Requerimientos funcionales

✅ El sistema debe permitir administrar categorías de productos (mostrar listado de categorías).
✅ Debe permitir gestionar productos organizados por categorías (filtrar productos al seleccionar una categoría).
✅ Los productos deben mostrar: nombre, descripción, precio e imagen representativa (emoji o ícono).
✅ El usuario debe poder agregar productos al carrito de compras.
✅ El carrito debe mostrar: productos agregados, cantidades, subtotales y total general.
✅ Se debe poder modificar cantidades (incrementar/decrementar) y eliminar productos del carrito.
✅ Opciones de "Continuar comprando" y "Completar compra" (simular finalización con un alert o mensaje de resumen).
✅ Sin base de datos · Los datos de categorías y productos deben estar precargados en un array dentro del estado de React.
✅ El carrito debe persistir al recargar la página usando localStorage.
